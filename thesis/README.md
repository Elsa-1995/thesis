# Автоматизация UI и API тестов для курсовой работы

##  Описание проекта
Этот проект автоматизирует тесты из курсовой работы по ручному тестированию. Проект содержит:
- 5 UI тестов для сайта Aviasales с использованием Page Object Pattern
- 5 API тестов для проверки API Aviasales
- Конфигурационные файлы для управления настройками
- Полную документацию по установке и запуску

##  Требования
- Python 3.8 или выше
- Google Chrome
- Доступ в интернет
